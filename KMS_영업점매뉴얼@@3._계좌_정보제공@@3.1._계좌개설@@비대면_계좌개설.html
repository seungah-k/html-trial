개요

1. 청년형 장기펀드계좌의 경우 소득확인증명서(만 34세 이상인 경우 병적증명서 추가 제출) 제출이 필수임에 따라 해당 서류의 진위여부 확인 후 이용 가능

2. 소득확인증명서, 병적증명서 진위 확인 업무 권한은 Digital고객관리본부에 부여.

신분증 진위확인

모바일 계좌개설시 신분증 진위확인이 완료된 개설신청건만 청년형 장기펀드 계좌번호 채번이 가능

비대면 계좌

개설상태 

조회

1. 『NH투자증권 QV MTS』 App과 『나무증권 MTS』 App을 통해 계좌 개설한 내역은 [4-4554] 화면에서 조회 가능

 1) 계좌개설신청 : 계좌개설 신청완료 후 출금계좌로 등록된 금융기관을 통한 이체입금 전 단계 (※ 통신 오류로 인한 신청계좌와 완료 건수가 불일치 하는 경우 등 포함)

    ☞ 관리자 승인 불가

 2) 이체입금완료 : 본인 명의의 금융기관(은행/증권)의 입력한 계좌번호로 당사가 1원입금 및 난수 확인이 완료된 상태

 3) 관리자 확인 : 관리자에 의해 비대면 실명확인을 하여 계좌개설 승인 및 거부로 개설업무 종료된 상태

    ☞ 거부여부 Y (소득/나이요건 미충족, 네트워크 장애로 인한 개설정보 오류, 고객요청)

        거부여부 N (증명서 진위여부 정상) 

 4) 신분증 재등록요청 : 신분증 이미지 저장실패로 인해 MTS 신분증 재등록 요청 상태

    ☞ 신분증 재등록 가능 기한 : 계좌개설 신청일 다음날부터 기산하여 휴일 포함 30일까지

    ※ 신분증 재등록 가능 경로 : MTS > My·고객센터 > 계좌개설 > 신분증재촬영

 5) 증명서 확인 필요 : 청년형 장기펀드 계좌개설 신청 후 Digital고객관리본부의 소득확인증명서 진위확인 필요 상태

 6) 증명서 재등록 요청 : 청년형 장기펀드 계좌개설 신청 후 고객이 등록한 증명서 발급번호 진위확인 불가로 고객에게 증명서 재등록 요청 상태

    ☞ 증명서 재등록 가능 기한 : 증명서 재등록 요청일+1영업일

Question 모바일 계좌개설


Answer:


Question 휴대폰 본인 인증 후 “나는 대한민국 출생이며 납세의무가 ~”에러가 날 경우 어떻게 해야 하나요?


Answer:

신규고객은 해당사항이 없으며, 기존고객의 경우 당사 계좌정보에 거주구분이 미등록 고객 일 경우 발생하는 에러입니다. 고객과 유선으로 본인 확인 및 국내거주여부를 확인하시어 [4-2040]종합계좌정정 화면에서 국내거주 여부를 선택하시면 됩니다.

※징구서류 : [징구용]고객정보변경 및 부가서비스신청서 (유선접수시 서류는 직원이 작성하여 보관)

※사유 : 비대면개설시 거주 여부 미등록으로 인해 개설 불가함에 따른 정보 정정.

Question 당일에 계좌개설을 중복해서 신청 가능한가요?


Answer:

불가합니다.

단기가 다수계좌 적용으로 당일에 중복 신규종합계좌개설이 불가합니다. 단, 빠른개설을 통한 개별계좌개설은 가능합니다.

Question 당사에서 비대면계좌개설을 여러번 하는것도 단기다수에 해당되나요?


Answer:

네, 해당됩니다. 

※ 당사 종합매매 계좌를 여러개 개설하시려는 경우, 비대면계좌개설 후 해당 계좌를 근거로 빠른계좌개설 하셔서 이용하실 수 있습니다. 

Question 출금기관 입력 시 “수행이 불가합니다. 012. 농협회원조합 계좌번호입니다.”일 경우 어떻게 해야 하나요?


Answer:

농협은행 및 지역농축협의 출금기관명이 해당 금융기관과의 계좌번호가 일치하지 않아서 나타나는 오류입니다.

해당 금융기관의 계좌번호가 맞는지 다시 확인하시어 등록하시면 됩니다.

Question 1원 본인인증시 외화통장 이용이 가능한가요?


Answer:

아니오, 외화통장은 본인인증용으로 기존계좌 활용이 불가합니다. 반드시 국내 본인 통장을 이용하셔야 합니다.


Question ISA 계좌개설 시 반드시 종합매매 계좌를 개설 후 만들어야 하나요?


Answer:

아닙니다. QVBranch 앱에서 ISA계좌개설로 바로 계좌개설신청이 가능합니다.

Question 비대면 계좌 승인 시 계좌명이 띄워 쓰기가 있는 경우 어떻게 해야 하나요?


Answer:

앱에서 고객이 성명을 수정할 수 없으며 최초 인증된 SMS 인증 확인 시 띄어쓰기로 입력되었고 한국신용평가에 고객데이터가 띄어쓰기로 되어있어 인증은 되었지만 실제 고객이 SMS인증시 띄어쓰기가 되었는지 또는 한국신용평가에 성과 이름이 띄어쓰기로 되어있는지 당사 전산에 파악이 불분명한 사항으로 고객이 타 업무 (인증서 발급 및 재발급 등

기타업무)를 하실 경우 계좌명이 띄어쓰기로 되어있을시 업무처리가 불가할 소지를 고려하여 고객확인 후 거부처리를 하셔야 합니다.



Question 관리자 거부 시 고액 책임자 승인창이 조회되면서 환불 처리가 되지 않아요!


Answer:

해당 고객이 다른 계좌에서 고액 출금이 있을 경우 해당점포의 책임자 승인[2-5515]처리 후 환불처리 하시면 됩니다.

Question [4-4554]의 발급일자, 면허번호와 [4-4555]의 신분증 추출정보가 상이할 경우 승인해도 되나요?


Answer:

[4-4555]비대면 계좌개설 관리 화면에 신분증 전체 이미지 기준으로 해당 고객의 성명, 주민번호, 성별, 내외국인 구분의 정보가 일치한다면 승인 가능합니다.

신분증 추출정보의 발급일자와 이미지는 해당이미지(좌측)의 추출정보로 일치 하지 않을 수 있습니다.



Question 계좌거부 후 입금했던 돈은 어떻게 되나요?


Answer:

계좌거부신청을 하면 고객이 지정했던 출금계좌로 자동 환불 처리됩니다.

하지만, 관리자 승인 전 지정 계좌 외 타행으로 입금되었다면 환불처리가 불가하며 고객이 지점 내방하여 실명확인 후 거부(폐쇄) 가능 합니다



Question 중개형ISA 계좌개설 시 반드시 종합매매계좌를 개설 후 만들어야 하나요?


Answer:

아닙니다. 앱에서 중개형ISA 계좌개설로 바로 계좌개설 신청이 가능합니다.

Question 앱에서 ISA계좌개설시 “한도 2000만원이 넘었습니다.”


Answer:

재형저축가입자의 경우 현재 앱에서 한도가 여유가 있음에도 불구하고 계좌개설이 되지 않고 있으며, 영업점에 내방하여 계좌개설을 하셔야 합니다.

Question ISA 계좌개설 시 최종 단계에서 개설신청버튼을 누르면 “고객정보가 존재하지 않습니다.”


Answer:

ISA는 랩계좌유형으로 상담FP가 등록이 되어야 합니다.

고객이 선택한 지점에는 상담FP가 지정되어있지 않는 지점으로 상담 FP가 지정되어 있는 

지점으로 변경 후 계좌개설신청이 가능합니다.


Question ISA 계좌개설신청에서 관리자 등록이 가능한가요?


Answer:

앱에서는 관리자등록은 불가하며, 상담 FP 와 투자권유대행인만 등록 할 수 있습니다.

Question 앱에서 투자권유대행인에 직원명을 넣었는데 계속 에러가 나옵니다.

Question 외화통장 이용이 가능한가요?


Answer:

출금계좌지정은 외화통장이 불가능합니다.

외화통장은 당사 자금통장을 통한 입출금 방식으로 본인인증 절차로는 적합하지 않습니다.

반드시 국내 본인 통장을 이용하셔야 합니다.



Question CD,ARS,영업점 창구에서 이체 가능한가요?


Answer:

CD, ARS, 영업점 창구에서는 소액이체가 불가능합니다.

대리인이 송금할 여지가 있어 본인인증방식에 해당되지 않아 반드시 인터넷 뱅킹 및 모바일 뱅킹을 이용하셔서 이체 하셔야 합니다.



Question 소액이체 이하로도 입금이 가능한가요?


Answer:

종합매매, CMA RP는 소액이체 1,000원 이하로는 입금이 불가능합니다.

ISA는 소액이체 가능 금액이 100,000원 입니다.

Question 비대면 계좌 승인 시 계좌명이 띄워 쓰기가 있는 경우 어떻게 해야 하나요?


Answer:

앱에서 고객이 성명을 수정할 수 없으며 최초 인증된 SMS 인증 확인 시 띄어쓰기로 입력되었고 한국신용평가에 고객데이터가 띄어쓰기로 되어있어 인증은 되었지만 실제 고객이 SMS인증시 띄어쓰기가 되었는지 또는 한국신용평가에 성과 이름이 띄어쓰기로 되어있는지 당사 전산에 파악이 불분명한 사항으로 고객이 타 업무 (인증서 발급 및 재발급 등 기타업무)를 하실 경우 계좌명이 띄어쓰기로 되어있을 시 업무처리가 불가할 소지를 고려하여 고객확인 후 거부처리를 하셔야 합니다.



Question 관리자 거부 시 고액 책임자 승인창이 조회되면서 환불 처리가 되지 않아요!


Answer: 해당 고객이 다른 계좌에서 고액 출금이 있을 경우 해당점포의 책임자 승인[2-5515]처리 후 환불처리 하시면 됩니다.

Question [4-4554]의 발급일자, 면허번호와 [4-4555]의 신분증 추출정보가 상이할 경우 승인해도 되나요?


Answer:

[4-4555]비대면 계좌개설 관리 화면에 신분증 전체 이미지 기준으로 해당 고객의 성명, 주민번호, 성별, 내외국인 구분의 정보가 일치한다면 승인 가능합니다.

신분증 추출정보의 발급일자와 이미지는 해당이미지(좌측)의 추출정보로 일치 하지 않을 수 있습니다.


Question 계좌거부 후 입금했던 돈은 어떻게 되나요?


Answer:

계좌거부신청을 하면 고객이 지정했던 출금계좌로 자동 환불 처리됩니다.

하지만, 관리자 승인 전 지정 계좌 외 타행으로 입금되었다면 환불처리가 불가하며 고객이 지점 내방하여 실명확인 후 거부(폐쇄) 가능 합니다.



Question 공동인증서 등록 하려는데 등록이 되질 않습니다.


Answer:

계좌개설 신청 후 관리자 승인이 되야 공동인증서 등록이 가능합니다.

관리자 승인 전 매매를 원하시는 고객은 간편비밀번호 등록 후 가능합니다. (MTS 메뉴 → 인증센터→ 간편인증번호)

Question 모바일otp발급시 업무제한 시간이 있나요?
